Maintaining a toaster computer: the HP ProBook 4520s

This is my beloved toaster. A machine I found in the garbage a few years back. It was mediocre machine when it was first launched. And it was almost unusuable when I found it.

Specifications:

  • Intel i3-370m @ 2.4 GHz 3M Cache, 2C/4T
  • 2GB DDR3 @ 1033 MHz
  • 300GB 5600 RPM HDD
  • 1366x768 WXGA Display
And that was only the start.

From this I knew I had a list of things to remedy:

  1. Reduce the disk swapping
  2. Cut boot times
  3. Replace the thermal paste
  4. Maybe see if a FHD panel is available

The first thing on my renovation list was the RAM. 2GB might have been enough back in 2007, but nowadays cell phones have more.

I ordered a 2x4GB Crucial RAM kit off of ebay, took out the battery, removed the speaker grill and keyboard and put the 2 new sticks in. Easy peasy.
